This is a simple implementation of the classic Tic Tac Toe game in Python.
Features

    Play against another player or against the computer.
    Randomly decides who starts the game when playing against the computer.
    Displays the current state of the board after each move.
    Checks for a winner after each move.
    Ends the game when a player wins or when all cells are filled without a winner (tie).

Installation:

    Ensure you have Python3 installed.

Setup:

    Clone the repository or download the tic_tac_toe.py file.

Run the Game:

    Open a terminal or command prompt.
    Navigate to the directory containing tic_tac_toe.py.
    Run the script using the command:

        python tic_tac_toe.py

    Gameplay:
        Choose whether to play against another player or against the computer (options 1 or 2).
        Enter the row and column numbers to place your mark ("X" or "O") on the board.
        The game alternates turns between players/computer.
        The game ends when a player wins by getting three of their marks in a row (horizontally, vertically, or diagonally), or when all cells are filled without a winner (tie).

Enjoy playing Tic Tac Toe!
